# Vite 매뉴얼 스캐폴딩

직접 Vite를 설치해 학습 환경을 재구성합니다.

<br />

## 사용 방법

```bash
git clone https://github.com/happyhye/vite-react.git
```
1. 리포지토리 clone

```bash
pnpm i
```
2. 패키지 설치

```bash
pnpm dev
```
3. 서버 구동

<br />

## 폴더 구조

![tree](https://github.com/user-attachments/assets/91950b83-7b64-40b8-af51-fcd35da8cc65)

- public
  - 정적요소 분리
- src
  - 컴포넌트, 스타일링(css) 코드 분리
- vite-scaffold
  - `degit` 명령을 사용하여 vite-react 프로젝트를 스캐폴딩

<br />
